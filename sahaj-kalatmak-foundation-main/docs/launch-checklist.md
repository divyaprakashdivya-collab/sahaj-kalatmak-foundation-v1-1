# Launch Checklist

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. Check that `index.html`, `people.html`, `events.html`, and `media.html` are at root level.
4. Enable GitHub Pages from Settings → Pages → Deploy from branch → main → root.
5. If using the custom domain, keep the `CNAME` file and verify DNS.
6. Test these URLs after deployment:
   - `/index.html`
   - `/people.html`
   - `/events.html`
   - `/media.html`
7. Confirm all five people images load from `assets/images/people/`.
